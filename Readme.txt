== Description ==
This slider give you advantage to show slider with use of shortcode:
[slider-shortcode img="" classname="custom class name"]




== Installation ==
1. Upload  WP tinymce editor slider shortcode  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In editor section a button get added named  "Add Slider images"
4. A pop up appears to upload images, when "tinymce editor" button is clicked
5. Once images get uploaded a new short code is generated. (Multiple images can be added at a time)
6. Generated short code can be used anywhere in the editor
